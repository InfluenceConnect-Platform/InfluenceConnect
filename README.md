# InfluenceConnect

A two-sided marketplace connecting Brands and Influencers.

## Tech Stack
- Frontend: Next.js, Tailwind CSS
- Backend: Node.js, Express.js
- Database: MongoDB Atlas
- Payments: Razorpay

## Project Structure
- `/frontend` — Next.js application
- `/backend` — Node.js + Express API
